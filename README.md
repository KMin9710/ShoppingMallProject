- JDK

- MySQL Workbench

## 데이터 테이블 구성
> Member Table
> - 웹페이지 전반적인 기능들을 이용할 때 가입된 회원의 정보
> - 회원들을 각각 구분하고 상품을 주문하거나 장바구니기능 사용
> - 관리자와 고객을 구분하기 위해 Type을 Columns에 추가

> Product Table
> - 웹페이지에 등록된 각각의 상품들의 정보
> - Product Detail 부분에서 상품의 정보를 확인할 수 있다.

> Orderlist Table
> - 장바구니 역할을 하는 테이블
> - 각각의 Product들의 정보를 저장하는 형식
